hexo

A tool to make your blog as static html and push it to server.

hexo install

Use npm to install hexo.
Node.js and Git are needed.

##1.install hexo

npm install -g hexo-cli
hexo init <folder>
    initial the hexo in the folder.
cd <folder>
npm install

##2.hexo folder menu

_config.yml
package.json
scaffolds
source
    _drafts
    _posts
themes

##3.hexo command

hexo generate / hexo g  -  generate static html with blog file
hexo n "blog_name"  -  create a new blog
hexo clean  -  clean cache, usually follow by: hexo g
hexo server / hexo s  -  start the server

hexo d  -  push the hexo file to remote github

##4.connect to the remote github
first create a new repo in github.com
setting new repo as public for free. private type need github pro account 

configure the _config.yml
in the end of the _config.yml is the deploy session
setting:
    type: git
    repo: remote github repo address
    branch: master
